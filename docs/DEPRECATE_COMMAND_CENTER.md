# Deprecating Command Center

When archiving `command-center` / `command-center-v2`, add this to the top of README:

```markdown
> **Deprecated:** Superseded by **[Mission Control](https://github.com/lionelresnik/mission-control)**.
> No further updates. This repo is kept for reference and rollback only.
> Live demo: [mission-control demo](https://lionelresnik.github.io/mission-control/)
```

Steps:

1. Push final commit to old repo if needed
2. Add deprecation banner to README
3. GitHub → Settings → Archive repository
4. Update any bookmarks / MCP configs to Mission Control

Do **not** delete the old repo.
